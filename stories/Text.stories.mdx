# Text

`Text` component can be used as a wrapper of paragraph or `p` tag.

```js
import { Text, Color } from 'uikit';

const CardWithText = () => {
  return (
    <Box p={2} bg={Color.black}>
      <Text fontSize={2} color={Color.white}>
        This is Text example
      </Text>
    </Box>
  );
};
```

## Available CSS properties to use as Props

Text uses css props from `styled-system` internally. These available props can be categorized as-

- [ColorProps](https://styled-system.com/api/#color) related to Text color
- [SpaceProps](https://styled-system.com/api/#space) related to Text spacing
- [LayoutProps](https://styled-system.com/api/#layout) related to Text Layout
- [TypographyProps](https://styled-system.com/api/#typography) related to Text Typography
