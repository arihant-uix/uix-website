# Heading

`Heading` component can be used as a wrapper of Heading or `h1 | h2 | h3 | h4 | h5 | h6` tag.

```js
import { Heading, Color } from 'uikit';

const CardWithText = () => {
  return (
    <Box p={2} bg={Color.black}>
      <Heading level={1} color={Color.white}>
        This is Heading example
      </Heading>
    </Box>
  );
};
```

## Available CSS properties to use as Props

Heading uses css props from `styled-system` internally. These available props can be categorized as-

- [ColorProps](https://styled-system.com/api/#color) related to Heading color
- [SpaceProps](https://styled-system.com/api/#space) related to Heading spacing

levels are same as Heading level, level 1 is similar to `h1` and so on.
